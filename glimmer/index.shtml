Glimmer 1.02
Glimmer is a tool useful to predict genes in genomic sequences.
The version of Glimmer you can use on this web page is 1.02
Please, paste or upload the genome sequence in FASTA format This is the genome sequence in FASTA format whose ORFs you want to find
Glimmer will locate long ORFs, extract them into a training set, then build a predicting system using IMMs (Interpolated Markov Models) and finally use this prediction system over the original genomic sequence to predict the genes.


For publication of results, please cite:
Valverde, J. R. (2009) http://www.es.embnet.org/Services/MolBio/glimmer1/
