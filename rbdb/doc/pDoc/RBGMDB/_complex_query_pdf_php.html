db_complex_query_pdf (line 69)

Generate a PDF report out of a user query

This function will connect to the specified database manager system using the ID and password provided, open the database (which we should have access to), run the user supplied query and generate a PDF report sorted according to user criteria.

The report will be returned directly to the user with the appropriate MIME type generated by the auxiliary library fPDF.

void db_complex_query_pdf (string $db_host, string $db_user, string $db_password, string $db_name, array $request, string $how)
  • string $db_host: the DBMS host
  • string $db_user: the user to connect to the DBMS as (should have access to the database
  • string $db_password: the user's password to gaina access to the DBMS
  • string $db_name: the name of the database
  • array $request: an associative array containing the query fields, keys and boolean algebra combinations
  • string $how: how to sort the resulting output
