Welcome to the Retinoblastoma Gene Mutation Database

You are now able to modify the database contents.

Please, be careful.

What would you like to do next?

  • Update the database contents.
    This allows you to substitute (i.e. delete) ALL current database contents and substitute them by a new dataset loaded from a file on your disk
  • Add a new record
    This allows you to add a new record to the database. You will be offered a form with spaces to fill all required data. Some minimal consistency checking will be done.
  • Edit a record
    To modify a record, please, use first the query forms to display it and then click on its edit button at the end of the table.
  • Delete a record
    To delete a record, please, use first the query forms to display it and then click on its delete button at the end of the table.
