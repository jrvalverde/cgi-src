WWW Homepage Access Counter and Clock
  • By: Muhammad A Muquit
  • Price: Freeware
  • Download it from Here! (1MB Unix Tar File) After downloading to your PC, FTP it as binary to a temp directory on your unix server.
Here it is! A really great counter and clock script just like the big sites use. Best of all, it is free. This program is great, but be forwarned, you need to read the instructions ahead of time and know the answers to important questions, such as your site's IP address.

Everyone has been asking me to find a good counter that does not require SSI, so here is a great one! The installation does not lend itself to e-mail handholding, so please try to figure it out. ;->

After moving this tar (tape archive) file to your /tmp directory, issue the command:

tar xvf wwwcount2_4.tar

It will create a new directory: wwwcount2.4
All instructions are in the docs file including pretty good set up instructions in html format. READ THE INSTRUCTIONS BEFORE ATTEMPTING INSTALLATION! You must have a "C" compiler on your system. You must also be able to access your site using telnet to install and compile this program.
