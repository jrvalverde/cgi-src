ENIAC Emulator

This page allows you to run programs on an emulated ENIAC system. The emulator provided is the same described in the CMPUT379 course from the University of Alberta. You can get more details regarding its implementation and the assembly language employed to program the emulator on the Assignment 01 of this course.
A manual of ENIAC programming was once available on the Net and can still be recovered from the Internet Archive.
To use the emulator upload or paste your program in the specified ENIAC assembler and submit it for translation and execution together with the input data. The results will be returned in both, binary and textual forms.
