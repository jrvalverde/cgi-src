ProFit

Protein Least Squares Fitting

ProFit

ProFit [1] (pronounced Pro-Fit, not profit!) uses McLachlan fitting algorithm, essentially a steepest descents minimisation as described in McLachlan, A.D. (1982) Rapid comparison of Protein Structures, Acta Cryst. A38, 871-873. ProFit is designed to be the ultimate program for performins least squares fits of two protein structures. The following paragraphs address the functionality of ProFit that is available from THIS service.

ProFit requires as input two structures: the reference structure that remains static in space and the mobile structure which is fitted onto it.

ProFit addresses the question of sorting out equivalent atoms by doing a sequence alignment using the Needleman-Wunsch algorithm. It then performs a least squares fitting of the equivalent residues in the two structures using N, C-alpha, C and O atoms, and produces the following reports:

  • A summary of the run describing what has been done and including
    • The Needleman-Wunsch alignment used to guide the fit
    • The summary of RMS deviation between both structures
    • The Centers of Gravity of the structures
    • The space transformations used for the fit
  • A detailed RMS calculation describing the by residue RMS deviation
  • The fitted mobile structure in PDB format
  • A PostScript representation of the fitted molecule
  • A VRML 2 model of the fitted molecule.

Please note that the program is called ProFit --- not PROFIT, Profit or profit; this attempts to avoid confusion with the threading program known as PROFIT. ProFit was written first and released to the public around the same time.

ProFit is able to perform more powerful and detailed analysis. To access its full functionality you should use it with the command line interface
