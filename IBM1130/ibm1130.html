IBM 1130 Emulator

This page allows you to run programs on an emulated IBM 1130 system with DMS R2v1.2 and the FORTRAN compiler. The emulator provided is the one provided by SIMH, a highly portable, multi-system simulator offered by the Computer History Simulation Project.
To learn more about the system, you may start at IBM1130.Org or IBM1130.Net or Answers.com. Many manuals are available from the Bitsavers archive.
